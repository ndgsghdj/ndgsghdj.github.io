# blog_wip

Writeups-only static blog generator for GitHub Pages.

Start here:

- [`docs/user-guide.md`](docs/user-guide.md) for the full guide
- `npm run build` to generate the site
- `npm run preview` to serve `dist/` locally

Main folders:

- `writeups/` source markdown
- `components/` reusable shortcode components
- `templates/` page shells
- `styles/` global stylesheet
- `images/` writeup assets
